# Projekt — rozcestník (Obsidian home)

Domovská nota znalostní báze hry **Důkazní materiál 1930**. Odsud vede cesta ke
všemu; v Obsidianu si ji nastav jako výchozí notu.

## Zdroje pravdy (ruční, autoritativní)

- [[../design-dokument|Design dokument]] — vize hry (jediný zdroj pravdy pro vizi).
- [[../prototyp-mvp|Prototyp / MVP]] — definice MVP, resoluční systém, metriky.
- [Herní obsah](../obsah/) — `karty.yaml`, `uzly.yaml`, `cile.yaml` (strojový zdroj pravdy).
- [[../prompty/protokol|Prompt protokolu]] — kanonické znění promptu (nikde jinde neupravovat).
- [Playtesty](../playtesty/) — poznámky ze sezení, jeden soubor na sezení.

## Procesní paměť (živá)

- [[stav|Stav projektu]] — roadmapa, backlog, otevřené otázky (udržuje `project-manager`).
- [[rozhodnuti|Log rozhodnutí]] — datovaná rozhodnutí a jejich důvod (append-only).

## Tým agentů

Definice v [`.claude/agents/`](../.claude/agents/). Model spolupráce: hybrid —
